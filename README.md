# Helpdesk CRM

Fullstack-приложение для управления тикетами поддержки.

## Фичи
- Создание, редактирование, удаление тикетов
- SLA-калькулятор (расчёт времени решения)
- Фильтрация и поиск
- Дашборд с аналитикой (Chart.js)
- Адаптивный дизайн
